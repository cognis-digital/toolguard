*AI Agents & LLMOps — build, route, evaluate, and secure agentic systems.*

## Why

TOOLGUARD exists for one job — runtime allowlist and policy for agent tool-calls — and does it without a SaaS bill or heavyweight setup.
Single-purpose, scriptable, CI-friendly, self-hostable, and callable by AI agents over MCP.

## Install

```bash
pip install cognis-toolguard
# or from this repo:
pip install -e ".[dev]"
```

## Quick start

```bash
toolguard --version
toolguard scan .                      # scan the current project
toolguard scan . --format json
toolguard scan . --fail-on high       # non-zero exit for CI gates
toolguard mcp                         # expose as an MCP server (Cognis.Studio / Claude Desktop / Cursor)
```
